# Paisabazaar-Customer-Credit-Risk-Analysis-using-EDA
This project analyzes customer financial and credit behavior using the Paisabazaar dataset through Exploratory Data Analysis (EDA) and data visualization techniques. The analysis includes data cleaning, outlier handling, and visualization using Python libraries such as Pandas, Matplotlib, and Seaborn.
# Customer Credit Score and Financial Behavior Analysis using EDA

## Project Description
This project analyzes customer financial and credit behavior using the Paisabazaar dataset through Exploratory Data Analysis (EDA) and data visualization techniques.

---

## Problem Statement
Financial institutions manage large amounts of customer financial data. This project analyzes customer credit behavior, debt, income, and payment patterns to identify factors affecting credit scores and financial stability.

---

## Business Objectives
- Analyze customer credit behavior
- Identify high-risk customers
- Improve credit risk management
- Support better loan approval decisions
- Understand relationships between financial variables

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## Data Visualization
The project includes:
- Pie Charts
- Bar Charts
- Heatmaps
- Scatter Plots
- Pair Plots
- Box Plots
- Correlation Analysis

---

## Key Insights
- Higher income customers generally maintain better credit scores.
- High debt and delayed payments are linked to poor credit scores.
- Customers with longer credit history manage debt more effectively.
- High credit utilization may indicate financial stress.

---

## Conclusion
This project helps understand customer financial behavior and supports better credit risk analysis, customer segmentation, and business decision-making using data-driven insights.
